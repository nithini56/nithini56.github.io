2. StudyGo App

Project information

  • Category:Mobile Application
  • Client: Tution Class Institute

Mobile Aplication for Tution Class

A mobile app for Tuition Class Management is designed using Flutter and Firebase. Student information management, attendance tracking, payment, report, and SMS automation for sending messages will become simple with the app. As a tool for an administrator, teachers, and students, it will serve as one platform for efficient academic and financial administration. Student registration, attendance tracking, payment, report, and real-time notification are key parts of the app. Integration with Firebase will make secure authentication, real-time storing, and scalability possible.

5. Grama Niladhari Online Appointment System

Project information

  • Category:Grama Niladhari Online Appointment System

The Grama Niladhari Online Appointment System is a web-based platform designed to streamline the appointment booking process for Grama Niladhari services.
